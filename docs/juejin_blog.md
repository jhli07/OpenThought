## ❓ 问题的开始

在某个深夜，我开始思考一个问题：

> "AI 能帮助人类思考吗？"

现在的 AI（ChatGPT、Claude 等）擅长的是**给答案**。你问问题，它回答。

但真正的思考往往不是这样的。

**真正的思考需要追问。**

这让我想起了**苏格拉底**。他从不直接给学生答案，而是通过提问，让学生自己找到答案。

这被称为"**苏格拉底式追问法**"。

---

## 💡 灵感闪现

如果我用 AI 来实现苏格拉底式追问呢？

不是让 AI 回答问题，而是让 AI **问更好的问题**。

当我有一个想法但不够清晰时，AI 可以：
- 抓住核心："什么让你想创业？"
- 深入探索："赚钱对你意味着什么？"
- 换个角度："如果失败了，你会失去什么？"

通过层层追问，帮助我真正想清楚一件事。

**这就是 OpenThought 的初衷。**

---

## 🎯 OpenThought 是什么？

OpenThought 是一个**深度思考工具**，它结合了：

1. **AI 的能力** - 大语言模型的智能
2. **苏格拉底追问法** - 经典的思想引导技术
3. **友好的界面** - CLI + Web 两种使用方式

它不是一个"回答问题的 AI"，而是一个"帮你思考的 AI"。

---

## ✨ 核心功能

### 1️⃣ AI 智能追问

OpenThought 会根据你的回答，动态生成下一个问题：

```
Q: 什么让你想创业？
A: 因为我想赚钱

Q: 赚钱对你意味着什么？
A: 意味着自由

Q: 自由对你来说意味着什么？
A: 意味着可以做自己想做的事
```

### 2️⃣ 完全开放

支持**任何**兼容 OpenAI API 的服务：

| 类型 | 提供商 |
|------|--------|
| 云服务 | OpenAI, Claude, Kimi, Qwen, DeepSeek, Zhipu, Yi, Minimax |
| 本地服务 | Ollama, LM Studio, LocalAI, vLLM |

你甚至可以连接自己的模型服务器！

### 3️⃣ 多重界面

- **CLI**: 终端交互，程序员的的最爱
- **Web**: Streamlit 界面，浏览器中使用
- **API**: 作为 Python 库集成到你的项目中

### 4️⃣ 思考追踪

保存你的思考历史，随时回顾：
- 完整的问答记录
- 洞察提取
- 导出分享

## 🚀 快速开始

### 安装

```bash
pip install openthought
```

### 交互式使用

```bash
openthought
```

### 代码使用

```python
from openthought import OpenThought

# 开始思考
ot = OpenThought(prompt="我想创业")
print(ot.think())  # AI 生成的问题
ot.ark("因为我想赚钱")
print(ot.think())  # 下一层追问
```

### 连接自定义 LLM

```python
from openthought.providers import create_provider

# 使用 Ollama 本地模型
provider = create_provider("ollama", model="llama3")

# 使用国产模型
provider = create_provider("qwen", "your-api-key")
```
