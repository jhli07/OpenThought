## 🎯 核心信息

**Slogan:** 用问题引导思考，用思考创造价值

**一句话介绍:**
> OpenThought - 一个用 AI 帮你深度思考的工具。不是给你答案，而是问你更好的问题。

**完整介绍:**
OpenThought 是一个开源的深度思考工具，它结合了 AI 的能力和苏格拉底式追问法，帮助用户对任何问题进行深入思考。无论是创业、决策还是人生困惑，OpenThought 都能通过层层追问，引导你找到内心真正的答案。

# 开源项目推荐

## 🧠 OpenThought - 用 AI 进行深度思考

> "提问本身，就是一种思考。" — Agent_Li

### 🎯 项目是什么？

OpenThought 是一个**深度思考工具**，它结合了 AI 的能力和苏格拉底式追问法，帮助用户：

- 把模糊的想法变成清晰的思考
- 通过层层追问找到内心真正的答案
- 建立更系统的思维方式

### ✨ 核心特性

1. **AI 智能追问** - 不是给你答案，而是问你更好的问题
2. **完全开放** - 支持任何 OpenAI 兼容的 API
   - 云服务：OpenAI / Claude / Kimi / Qwen / DeepSeek / Zhipu / Yi / Minimax
   - 本地服务：Ollama / LM Studio / LocalAI / vLLM
3. **多种界面** - CLI 终端 + Streamlit Web 界面
4. **生产级代码** - 完整测试、文档、CI/CD

### 🚀 快速开始

```bash
pip install openthought
openthought "我想创业"
```

### 💡 使用场景

- 创业决策
- 职业规划
- 人际关系
- 人生思考
